# Fast Food Sales Performance Dashboard

## Overview
This project analyzes fast food sales data to uncover trends in product demand, revenue contribution, and monthly transaction patterns. An interactive Tableau dashboard was built to support data-driven insights.

## Tools Used
- Tableau
- Excel

## Objectives
- Identify top-selling items by quantity  
- Compare revenue contribution across item types  
- Analyze monthly transaction trends  

## Key Insights
- Cold Coffee leads in total quantity sold.  
- Sugarcane Juice and Panipuri show strong sales volume.  
- October shows the highest overall transaction value.
